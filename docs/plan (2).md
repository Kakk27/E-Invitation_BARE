# 🎩 Corporate Gala — E-Invitation Plan

> **Event Type:** Corporate Gala  
> **Format:** E-Invitation Website (Single Scroll Page)  
> **Document:** Timeline & Schedule Plan

---

## Overview

This plan covers the full timeline and schedule for building and launching the Corporate Gala e-invitation website — from initial setup through post-event follow-up.

---

## Phase 1 — Pre-Production (Weeks 1–2)

### Week 1 · Content Gathering

| Task | Owner | Due |
|------|-------|-----|
| Confirm event name, date, time, and dress code | Event Manager | Day 1 |
| Finalize venue name, address, and directions link | Event Manager | Day 2 |
| Write event description and programme of events | Copywriter | Day 3 |
| Collect host/company logo and branding assets | Design Lead | Day 3 |
| Source or produce intro video (15–30 sec) | Videographer | Day 5 |
| Define RSVP deadline and guest capacity | Event Manager | Day 5 |

### Week 2 · Design & Development

| Task | Owner | Due |
|------|-------|-----|
| Finalize color palette and typography choices | Design Lead | Day 8 |
| Build HTML/CSS/JS single-page invitation site | Developer | Day 10 |
| Integrate intro video with auto-dismiss logic | Developer | Day 10 |
| Build and test RSVP form with success state | Developer | Day 11 |
| Cross-browser and mobile responsiveness check | QA | Day 12 |
| Internal review and feedback round | All | Day 13 |
| Apply revisions from review | Developer | Day 14 |

---

## Phase 2 — Launch (Week 3)

### Week 3 · Go Live

| Task | Owner | Due |
|------|-------|-----|
| Deploy invitation site to hosting platform | Developer | Day 15 |
| Set up custom domain (e.g. gala2025.company.com) | Developer | Day 15 |
| Connect RSVP form to email/database backend | Developer | Day 16 |
| Test full user flow end-to-end | QA | Day 16 |
| Send invitation link to guest list via email | Event Manager | Day 17 |
| Send follow-up reminder (7 days before RSVP deadline) | Event Manager | Day 21 |

---

## Phase 3 — RSVP Management (Weeks 3–5)

| Task | Owner | Notes |
|------|-------|-------|
| Monitor RSVP submissions daily | Event Manager | Check email/dashboard |
| Send confirmation email to each respondent | Automated | Via form backend |
| Track attendance count vs capacity | Event Manager | Update weekly |
| Chase non-responders 3 days before deadline | Event Manager | Personal email/call |
| Close RSVP form on deadline date | Developer | Disable form submission |
| Export final guest list | Event Manager | Share with venue |

| Task | Owner | Due |

---

## Notes

- RSVP deadline should be set **at least 2 weeks** before the event for accurate catering and seating.
- The intro video should be **under 30 seconds** and **under 10 MB** for fast loading on mobile.
- Ensure the invitation link is **mobile-friendly** — most guests will open it on their phones.
- Consider adding a **calendar invite (.ics)** download link on the hero section.
